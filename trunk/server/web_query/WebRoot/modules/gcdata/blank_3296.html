范家田水库工程简介

 

范家田水库位于威宁县西北80公里的牛棚镇白碗村。水库集雨面积3.2平方公里，圬工重力坝，最大坝高12.6米，总库容77万立方米，设计灌溉面积5000亩，有效灌溉面积150亩。

水库设计洪水20年一遇，流量14立方米/秒；校核洪水100年一遇，流量31.3立方米/秒，坝顶溢流，坝址为石灰岩。

1968年动工，1971年建成，总投资5万元。因水库至今不配套，无灌溉渠道，一直未达到设计效益。
