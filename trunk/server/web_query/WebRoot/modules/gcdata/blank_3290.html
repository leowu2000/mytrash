赖毛冲水库工程简介

 

赖毛冲水库位于威宁县西北109公里的中水镇。水库集雨面积1.3平方公里，均质土坝，总库容54万立方米，设计灌溉面积4500亩,有效灌溉面积1000亩。干渠1条长3.5公里。

水库设计洪水20年一遇，流量12立方米/秒；校核洪水200年一遇，流量24.7立方米/秒，右岸开敞式溢洪道泄洪。

1959年动工，1960年建成，但无溢洪道，1982年对溢洪道进行配套。累计工程总投资13.6万元。
