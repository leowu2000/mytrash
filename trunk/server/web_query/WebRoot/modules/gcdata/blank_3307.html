深沟水库工程简介

 

深沟水库位于赫章县东面36公里的平山乡龙壮村。水库集雨面积1.7平方公里，均质土坝，最大坝高27米，总库容112万立方米，设计灌溉面积1200亩，有效灌溉面积972亩，干渠1条长2.3公里，支渠2条长4.2公里。

水库设计洪水30年一遇，流量6.2立方米/秒；校核洪水300年一遇，流量9.6立方米/秒，左岸开敞式溢洪道，坝址为土基。

1973年动工，1976年建成。八十年代末进行了坝身加固，渠系配套，总投资29.3万元。现在灌溉效益较好。
