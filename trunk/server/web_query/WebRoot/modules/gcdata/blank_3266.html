蓑衣塘水库工程简介

 

蓑衣塘水库位于织金县东北61公里的太平乡四角田村。水库集雨面积0.4平方公里，均质土坝，最大坝高10.5米，总库容13万立方米。设计灌溉面积500亩,有效灌溉面积353亩。干渠1条长1公里，支渠3条长1.7公里。

水库设计洪水20年一遇，流量8.5立方米/秒；校核洪水200年一遇，流量14.1立方米/秒，在库区右岸石灰岩地段修建暗渠式泄洪涵洞一道，断面1.0×1.5米，长10米。

1963年动工，1964年建成。1985年对坝身及库区右岸进行防治处理。总投资49.1万元。
