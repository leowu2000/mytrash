工程概况

    赫章县位于贵州省的西北部，属乌江水系六冲河支流的上游。地理位置在东径104010＇20＂至1050　01＇23＂，北纬260　46＇12＂至27028＇18＂全县为西高东低一斜坡地形，东西长84.3公里，南北长77.6公里，总面积为3245平方公里。

      县内以高原中山岩溶地貌为主，岩溶地貌发育齐全，地形地貌复杂，山高坡陡，河谷深切。县内第四纪沉积不太发育，从石炭系二叠纪的碳酸盐石和三叠纪飞仙关组紫色砂页岩广泛分布。

    赫章县龙场坝水库位于赫章县城东面的平山乡平山片区的 中山 村坝头组的龙场坝，距离县城32ｋｍ，地理坐标：东径104053＇17″，北纬27008＇00″。属乌江水系七星河支流上游。水库坝址位于一山顶向斜地带，周边属于平山国有林场，植被比较好，水土流失很轻微，因此该水库到现在为止还没有较大的淤积，坝址以上集雨面积0.4 ㎞ 2 ，该水库总库容17万ｍ3，属小（二）型水库。工程的兴建当时主要是为了解决下游500亩稻田的灌溉（当时设计灌溉面积），是一座以灌溉为主要功能的水库。水库建成后这水库就成了下游具有防洪责任的水库，主要任务还承担着保护下游500人的生命财产安全，150亩田地的防洪保护面积。
