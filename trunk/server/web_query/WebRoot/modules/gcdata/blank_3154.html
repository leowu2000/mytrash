松山大塘工程简介

 

   松山大塘水库位于毕节县西南45公里的杨家湾松山村。水库集雨面积0.7平方公里，均质土坝，最大坝高12米，总库容17万立方米,设计灌溉面积560亩，实际灌溉面积为322亩。干渠1条长0.3公里，支渠2条长1.2公里。

   水库设计洪水20年一遇，流量7.9立方米/秒;校核洪水200年一遇，流量14.9立方米/秒。右岸开敞式溢洪道。

     1973年动工，1974年建成，国家补助投资1.9万元
