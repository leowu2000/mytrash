小沟水库简介

    小沟水库位于金沙县安底镇八寨村，距安底镇5km,属长江流域乌江水系偏岩河上游的茶园河支流。地理位置东经106°23′28″，北纬27°16′23″，金沙至沙土水泥路擦肩而过。水库集雨面积6.1km2，水库总库容21.55万m3，防洪库容6.1万m3，属小二型水库，是一座以灌溉为主的水利枢纽工程。设计灌溉面积700亩，有效灌溉面积560亩。

    水库工程枢纽由大坝、溢洪道、放水系统等组成。水库总库容21.55万m3，属小二型水库工程，洪水标准按20年一遇设计，100年一遇校核。该水库大坝类型为浆砌石拱坝，坝顶高程1020.25m，坝长46.43m，坝顶宽2.9m，最大坝高20.2m。溢洪道为河岸开敞式。放水设施为三个∮300mm闸阀。
