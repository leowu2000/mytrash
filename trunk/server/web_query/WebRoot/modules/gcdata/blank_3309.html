 杨七沟水库简介

    杨七沟水库工程位于赫章县城东面的平山乡中山村，是赫章县的鱼米之乡，距县城44公里，水库承担着下游120亩稻田的灌溉任务，是一座以灌溉为主的水库。杨七沟水库设计标准采用20年一遇洪水设计，200年一遇洪水校核，水库下泄流量为5.3ｍ3/Ｓ；当超过20年一遇洪水时，除冲毁少量的部分田地外，几乎对水库及下游也造成不了多大灾害。杨七沟水库为均质土坝，最大坝高为16.9ｍ，坝顶高程1611.88ｍ，坝顶长73.00ｍ，坝顶宽3.0ｍ，大坝上游坡比为1：3.0，下游坡比1：2.5，溢洪道为左岸坡开敞式溢流，其布置在大坝左肩的山体间，溢洪道底高程1609.53米，溢洪道净宽2米。
    杨七沟水库所处区域地震基本烈度小于VI度，区域构造无复活现象，水库无诱发地震的可能性，区域稳定性好。库区无大的构造通过，无岩溶和临谷渗漏，库岸条件好。并且上游植被比较好，因此，库水水源含泥沙量比较小。但由于水库运行时间长，大坝目前主要有坝基、坝体中下部渗漏，现在该水库属于病险库,故该水库近几年几乎一直处于不能过高的水位运行状态。
