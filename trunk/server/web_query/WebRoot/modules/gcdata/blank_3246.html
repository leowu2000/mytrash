继红水库工程简介

 

继红水库位于金沙县东南37公里的文家桥水库上游2公里的黄泥堡。水库集雨面积2.9平方公里，浆砌石单曲拱坝，最大坝高20.8米，总库容28万立方米，灌溉面积350亩,有效灌溉面积100亩，干渠2条长2公里。

水库设计洪水20年一遇，流量14立方米/秒；校核洪水200年一遇，流量22.8立方米/秒，坝顶溢流，溢流段长度5.8米。坝址为中上寒武系灰岩。

1977年动工，1978年建成，总投资15.5万元。

由于岩溶较发育，该水库尚存在浇坝渗漏问题，需要灌浆处理，才能正常蓄水。
