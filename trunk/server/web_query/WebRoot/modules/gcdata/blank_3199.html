干河沟水库工程简介

 

干河沟水库位于黔西县东北52公里的金沙县宋坪干河沟。水库集雨面积23.5平方公里，坝型为浆砌石拱坝，最大坝高36.5米，总库容314万立方米，设计灌溉面积7042亩，有效灌溉面积达3500亩。坝后电站装机150千瓦。干渠3长26.4公里，支渠6条长21公里,

水库设计洪水30年一遇，流量136.9立方米/秒；校核洪水300年一遇，流量226.4立方米/秒，坝顶溢流，溢流段长度22米。

坝址为不对称“V”形河谷，地层下三迭系飞仙关组紫红色灰岩、砂岩、页岩，为适应不对称河谷，采用单曲双心拱形。

1975年动工，1980年至1982年停缓建，1983年恢复建设，1985年全部建成。总投资365.6万元。
