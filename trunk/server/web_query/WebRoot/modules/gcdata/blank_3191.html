杨家湾水库工程概况

工程概况

杨家湾水库属兴隆乡管理的小<二>型水库工程，位于大方县城东北角,距县城65km ,交通条件极差，水库不通公路。该工程位于石板村境内,于1977年动工兴建,1991年县水电局勘测设计,同年由瓢井区和县水利局抽人组建工程指挥部实施，建有土坝1座，水库总库容15万m3，因诸方因素工程未能发挥工程效益。由于放水涵洞在施工中处置不当,库水沿放水涵洞漏出，现不能蓄水，仅空库一座。
