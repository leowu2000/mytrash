红阳水库工程简介

 

红阳水库位于织金县西南32公里的后寨乡喇叭河，水库集雨面积24.3平方公里，坝型为土石合坝，最大坝高39米，总库容788万立方米，设计灌溉面积10251亩，达到有效灌溉面积8000亩。坝后电站装机容量2×100千瓦，为干渠沿线解决2乡1镇7118人，牲畜3798头的人畜饮水困难。干渠1条长30公里，支渠5条长17.7公里。

水库设计洪水50年一遇，流量191立方米/秒；校核洪水500年一遇，流量351立方米/秒，左岸侧槽式溢洪道泄洪。坝址及库区为三迭系永宁镇组灰岩，泥质灰岩，泥岩。

该工程1966年动工，1972年建成。1973年至1980年两次对大坝进行漏水处理和干渠修建，1987年至1990年列为水利部扶贫项目，又对大坝基础进行灌浆处理，迎水坡用塑料薄膜防渗及渠系配套，累计总投资663万元。

1992年5月，织金发生特大暴雨，红阳水库渠系多处发生垮塌、冲毁，使灌溉效益不能发挥，正拟修复
