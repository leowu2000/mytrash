吹聋水库工程简介

 

吹聋水库位于织织县西南50公里阿弓镇吹聋村。水库集雨面积1.6平方公里，但有一股地下水流入库区，主坝最大坝高18米，副坝2座7～9米，总库容122万立方米，设计灌溉面积2500亩，有效灌溉面积1200亩，干渠3条长10公里。

水库设计洪水30年一遇，流量37立方米/秒；校核洪水300年一遇，流量67立方米/秒，利用库区左岸垭口设开敞式洪道泄洪。

工程1958年动工，1981年建成，1982～1987年两次对大坝进行漏水处理，但该库区及坝址存在多层岩溶管道渗漏，虽然处理，尚未根冶，目前只能部份蓄水运用，进行溢洪道改建及渠道防渗，累计总投资113.9万元。
