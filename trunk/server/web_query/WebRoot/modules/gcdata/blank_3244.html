官田水库工程简介

 

官田水库位于金沙县东南68公里的官田乡。水库集雨面积3.1平方公里，均质土坝，最大坝高6.5米，总库容15万立方米,设计灌溉面积为540亩，有效灌溉面积为400亩。

水库设计洪水20年一遇，流量16.7立方米/秒；校核洪水200年一遇，流量27.1立方米/秒。右岸开敞式溢洪道，坝址为土基。

1958年动工，1960年建成，国家补助投资4.2万元。

九十年代初，对坝身进行了加固，溢洪道扩建，使水库能正常蓄水，安全运行。
