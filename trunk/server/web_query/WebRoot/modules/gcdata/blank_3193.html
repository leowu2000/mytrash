洪家渡水电站

    洪家渡水电站位于乌江上游支流北源的六冲河中下游，是乌江梯级水电站具有多年调节的龙头水库。

    洪家渡水电站水库正常高水位（正常蓄水位）1140m，相应库容44.97亿m3，死水位1076m，死库容11.36亿m3，总库容49.47亿m3，调节库容33.61亿m3。

    洪家渡水电站枢纽主要建筑物由混凝土面板堆石坝、洞式溢洪道、泄洪洞、引水系统、地面厂房、防渗系统组成，洪家渡混凝土面板堆石坝、洞式溢洪道和泄洪洞按一等一级标准设计建设，引水系统和地面厂房按一等二级标准设计建设，防渗系统按一等三级建设。混凝土面板堆石坝，坝顶高程1147.50m，最大坝高179.50m，坝顶长427.79m，坝顶宽度10.95m，坝底宽度510m，上下游综合边坡1：1.4。大坝宽高比为2.5：1，属狭窄河床高面板堆石坝；泄洪洞进口底板高程1056m，泄洪洞全长874.509m，竖井式闸门井内设一道6.8×9m（宽×高）平板检修门，有压段末段设一道弧型工作门，泄洪洞弧型工作门6.2×8m（宽×高），有压洞直径9.8m，平坡，无压隧洞段断面为7×12.621m（宽×高），城门洞型，纵坡为7.5%，出口采取曲面贴角鼻坎挑流消能；洞式溢洪道堰顶高程1122m洞式溢洪道进口为2孔10×18m（宽×高）的开敞式溢流堰，各设一道弧型工作门，堰后为平面收缩段，洞式溢洪道长861.72m，纵坡7.5%，断面14×21.54m（宽×高），城门洞型，出口采用曲面贴角鼻坎挑流消能；引水系统采用三洞三机单元供水方式，发电机单机额定流量165.50m3/s。地面厂房位于面板堆石坝脚左岸，安装三台（3×200MW）的水轮发电机组，水轮机组安装高程974.00m，发电机层高程988.00m。
