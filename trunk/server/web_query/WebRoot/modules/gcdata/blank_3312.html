 工程简介

        坝研水库位于距赫章县城31公里的平山乡和平山村，属小（二）型水库，水库所在流域为六冲河流域，水库有效灌溉面积550亩，总库容16万m3,根据《防洪标准》(GB50201—94)的工程等级主要建筑物大坝为四级，次要建筑物为五级。

   水库设计洪水采用20年一遇，流域1.5m3/秒；200年一遇洪水校核，流量3.2m3/秒。当超过20年一遇洪水时即为超标准洪水。坝研水库主要建筑物为均质土坝，最大坝高23m，坝顶高程1725m，坝顶长154m，坝顶宽3m，大坝上游坡比为1：2.5，下游坡比为1：2.0。溢洪道为左岸开敞式溢流，布置在大坝左侧的山体间，溢洪道进口处宽4m，中部宽2m，高1.2m，进口底高程1720m，最大泄流量3.4m3/s。放水设备位于大坝左端，为圆形钢管，高程进口1712.00m，断面尺寸￠150mm，流量0.10m3/S。灌溉引水干渠一条长2km，有效灌溉面积550亩。
