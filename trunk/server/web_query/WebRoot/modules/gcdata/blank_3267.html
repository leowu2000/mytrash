杠坝水库工程简介

 

杠坝水库位于黔西县西南80公里的白泥乡起马村。水库集雨面积1.0平方公里，均质土坝，最大坝高15米，总库容11万立方米，设计灌溉面积650亩,有效灌溉面积360亩。干渠1条长3公里，支渠1条长2公里。

水库设计洪水20年一遇，流量18.1立方米/秒;校核洪水200年一遇，流量21.1立方米/秒，右岸开敞式溢洪道泄洪，坝址为土基。

1958年动工，1959当年建成，总投资39.2万元。
