东风电站简介

    东风电站位于大方县东关乡岩下村,地理座标为：东经105°30′08″,北纬27°10′54″。距县城12km，老清毕公路从左坝肩上部半山穿过,交通较为便利。

    该工程为省水利水电勘测设计院设计，装机容量为2×1600kw，由大方县委组织施工，水电八局负责安装。1970年开始筹建，1978年元月14日竣工发电；建有浆砌块石重力拱坝一座,最大坝高30m,坝顶拱圈外弧长152.80m，坝顶溢流式,溢流堰顶宽77.55m,总库容1128万m3，其中：死库容522万m3,兴利库容624万m3,调洪库容504万m3；动力引水渠设在左坝肩,渠长101.30m,设计引用流量17.32m3/s,立式机组安装。

    电站建成后,由大方县水电局管理,1984年转由毕节地区电力局管理,属地区供电局直辖企业,现有职工51人,运行至今，各主体工程状况良好。
