新华水库工程简介

 

新华水库位于威宁县西北109公里的中水镇猫脑壳牛栏江上游西水河支流上。水库集雨面积42.5平方公里，坝型为浆砌石单曲拱坝，最大坝高49米，总库容462万立方米，设计灌溉面积18000亩，现到达有效灌溉面积7113亩，防洪保护面积3000亩，坝后电站装机2×50千瓦。干渠1条长8.7公里，支渠5条长23公里。

水库设计洪水20年一遇，流量214立方米/秒；校核洪水200年一遇，流量307立方米/秒。坝顶溢流，低鼻坎挑流消能。

坝址河床为不对称“V”形河谷，地层为二迭系峨眉山玄武岩。为改善坝身应力条件，在右岸设重力礅。

1975年动工，1983年建成。总投资453.3万元。
