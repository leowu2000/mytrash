群丰水库

群丰水库工程简介

 

群丰水库位于织金县南37公里的后寨乡麻窝寨村。水库集雨面积1.4平方公里，均质土坝，最大坝高18.5米，总库容40万立方米，设计灌溉面积500亩,有效灌溉面积200亩。干渠2条长7公里。

水库设计洪水20年一遇，流量26立方米/秒；校核洪水50年一遇，流量43立方米/秒，右岸开敞式溢洪道泄洪，坝址为土基。

1958年动工，1963年建成，1992年至1993年对大坝进行防渗处理。总投资49.1万元。
