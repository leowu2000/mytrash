云雾山水库工程简介

    云雾山水库位于威宁县城东部六十五公里，地处二塘镇新合村境内。大坝地理位置为东经104°42′15″，北纬26°44′10″，属长江流域乌江支流二塘大河。该水库大坝为均质土坝，总库容19.5万立方米，集雨面积2.6平方公里。放水管为D300毫米铸铁管，进行过多次更换，现运行良好。溢洪道为侧槽式，未进行衬砌海底，无消力池。

    云雾山水库原设计以灌溉为主，兼顾防洪。设计灌溉面积2250亩，其中田100亩，土2150亩。防洪保护面积1280亩，由于工程大坝完工后，未进行渠系配套，故毫无灌溉效益，现仅有1250亩的防洪保护面积。在防洪保护面积内有3个村3000人口，还有新合磷肥厂和大湾至贵阳铁路干线。
