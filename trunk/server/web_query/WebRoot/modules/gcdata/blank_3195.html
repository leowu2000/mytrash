沙坝河水库

    沙坝河水库位于贵州省黔西县中部野纪河干流上，东经105°47′－106°27′，北纬26°46′－27°21，大坝距黔西县城18公里。

    沙坝河水库是贵州黔西发电厂的供水水源工程，为完全多年调节中型水库，属Ⅱ等工程。工程始建于2003年，2005年5月下闸蓄水。坝址以上控制区集雨面积638平方公里，总库容4310万立方米，有效库容2050万立方米。有钢筋混凝土面板堆石坝一座，坝轴线NE60°，坝顶高程1181.5米，防浪墙顶高程1182.7米；最大坝高54.5米，坝顶长度169.061米，坝顶宽6米。溢洪道为岸边开敞式无闸控制自由溢流型式，溢流堰顶高程与正常蓄水位1174.5米相同。进水渠宽44.5米，长约96.31米，底板高程1172.0米。
