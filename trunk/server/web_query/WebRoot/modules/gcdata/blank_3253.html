大湾水库简介

    大湾水库工程位于金沙县城东面的龙坝乡境内，金沙县城距龙坝乡政府驻地15公里，可通车辆，本工程所在地位于龙坝乡政府南面，距乡政府3.5公里，不通车。地理位置东经106°24′54″，北纬27°22′12″。水库集雨面积0.5km2，水库总库容23.04万m3，防洪库容1.7万m3，属小二型水库，是一座以灌溉为主的水利枢纽工程。设计灌溉面积1206亩，有效灌溉面积1000亩。

    大湾水库工程枢纽由大坝、溢洪道、放水系统等组成。水库总库容23.07万m3，属小二型水库工程，洪水标准按20年一遇设计，100年一遇校核。该水库大坝类型为拱坝，坝顶高程1115.85m，坝长56.1m，坝顶宽1.5m，最大坝高23.08m。溢洪道为河岸开敞式。放水设施为∮300mm闸阀。
