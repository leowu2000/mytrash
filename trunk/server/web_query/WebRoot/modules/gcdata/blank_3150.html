小和平水库工程简介

 

小和平水库位于毕节县东北87公里的生机乡小和平村。水库集雨面积1.7平方公里，均质土坝，最大坝高30.5米，总库容46万立方米，干渠2条长7.2公里，支渠3条长9.1公里，设计灌溉面积2500亩,有效灌溉面积1260亩。

水库设计洪水20年一遇，流量13.4立方米/秒；校核洪水200年一遇，流量21.7立方米/秒。

1972年动工，1976年建成。国家补助投资10万元。
