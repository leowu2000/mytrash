龙滩口水库

    龙滩口水库位于黔西县谷里镇建全村，距镇所在地2.5公里，其地理座标约东经106°11′,北纬27°00′,该水库流域属于乌江水系。地层沿黔西向斜核部分布，地貌上为溶丘谷地，谷底开阔，地势平缓，属于中低山地带，气候属中亚热带温润气候区，流域内植被良好，水土流失较为轻微；库区流域内集雨面积0.16km2，年径流总量19.3万m3；工程设计灌溉面积300亩，有效灌溉面积300亩。

    水库来水基本由大气降水及泉水补给，排水均沿着地表、裂隙、孔隙及洼地、漏斗、溶洞渗入地下，补给地下水，水库区域内无水文监测站，也无雨水情自动测报系统。

    龙滩口水库始建于1956年，位于黔西县谷里镇，是谷里镇主要的防洪灌溉工程之一。该水库为小（二）型水库，水库设计总库容11.10万m3，兴利库容9.35万m3，死库容0.15万m3。现建有土石混合坝一座,坝顶高程1242.06m，最大坝高7.76m，坝顶长34.8m，顶厚27.0m，溢洪道设置在左坝端,长36.0m,宽1.5m。正常蓄水位1241.206m，设计洪水位1241.611m，校核洪水位1241.773m，死水位1235.32m。上游坝坡为1:0的浆砌石垂直墙，下游坝坡为1：4；溢洪道为开敞式自然溢流堰，堰底高程为1241.206m，设计下泄流量为1.0m3/s，放水设备原为涵卧管，涵管为方形，后改为涵洞内铁管闸阀，底孔高程1234.74m。
