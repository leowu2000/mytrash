红旗水库工程简介

 

红旗水库位于纳雍县西北59公里的猪场乡大多拱。水库集雨硇20.5平方公里，坝型为土石混合坝，最大坝高26.8米，总库容400万立方米，设计灌溉面积11587亩，干渠1条长36.5公里，支渠5条长23.2公里，利用渠道跌水建电站1座，装机2×75千瓦，坝后电站一座，1×50千瓦。

水库设计洪水50年一遇，流量274立方米/秒；校核洪水300年一遇，流量421立方米/秒，左岸开敞式溢洪道泄洪，坝址地层为下三迭宁镇组灰岩及泥质灰岩。

1958年动工，1960年停建，1967年恢复建设，1969年再次停建，1973～1975年续建完成至现有规模，建设期历时16年，累计总投资183.8万元。由于渠系长期不能配套，淤积垮塌严重，实际灌溉面积波及系数仅达490亩，坝后电站仍在发电，渠道跌水电站现已报废。
