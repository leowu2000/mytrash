绿堰滩水库工程简介

 

绿堰滩水库位于黔西县东北面43公里的中坪营镇吉丰村。水库集雨面积10平方公里，均质土坝，最大坝高19米，总库容14万立方米。设计灌溉面积650亩，有效灌溉面积600亩。干渠1条长6.8公里。

水库设计洪水100年一遇，流量80.2立方米/秒；校核洪水200年一遇，流量109.5立方米/秒。右岸开敞式溢洪道泄洪。

1957年动工，1981年建成,总投资17.1万元。

该水库坝身已有沉陷变形，拟进行加固处理。
