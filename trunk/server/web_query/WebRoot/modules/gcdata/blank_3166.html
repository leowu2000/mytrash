大消洞水库

 

大消洞水库地理位置为东经105°54′，北纬27°08′，于大方县城的东面，距县城38km，大坝建在黄泥塘镇芦塘村境内，水库集雨面积2.51km2，属六冲河流域。

大坝于一九六六年动工兴建，同年七月毕节地区农水局勘测设计，地区农水局审批，第一次国家补助经费12.46万元。由鸡场区、化联公社组建工程指挥部，组织民工施工，县农水局派技术人员作指导。大坝于一九七五年竣工，完成土石混合坝一座，最大坝高19.2m，总库容53.8万m3。

大消洞水库竣工蓄水后，发现坝基及右坝肩渗漏严重，主要渗漏点在放水涵洞出口附近，高程约1546m左右，当库水位达1551.6m左右，当库位达1551.6m，主要漏点的漏水量约8升/秒，随着水位的上升，漏水量也不断加大，水库不能正常蓄水。

一九七六年至一九八二年，县水电局设计，审定拔款4.74万元，修建溢洪道、管理房及3km的渠道防渗。由化联化社（乡）、鸡场水工站组织民工施工，一九八三年施工结束。至九一年底国家共投资17.2万元，群众自筹2万元。

由于工程无灌溉效益，故根据坝顶高程（1561.48m）、溢洪道高程（1560.06m）计算防洪限制水位。经计算当P=0.5％，QP＝19.8m3/S,Wp=43.5万m3，校核洪水位1560.77m，总库容53.8万m3，滞洪库容38万m3，溢洪道水深0.6m（渠道式），上游雍高值0.11m，防洪限制水位为1554.05m（相应库容15.8万m3，风浪爬高0.57m，安全超高0.2m。）

大坝的坝坡、顶宽已达到或基本达到规范要求，迎水坡只护至1551.65m高程。

水库右坝肩及坝基渗漏严重，溢洪道淤塞约0.5m高，放水闸门起闭机破坏，闸门不能开启，渠道淤塞严重，管理房被破坏，因而目前为一废水库。

大消洞水库设计灌溉面积2077亩，由于坝基及坝肩严重渗漏，水库不能正常蓄水，灌区土变田又均未实现，故目前未发挥效益。

     从九十年代初,己报废，至今,未发生过险情。
