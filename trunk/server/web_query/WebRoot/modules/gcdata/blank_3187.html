小海坝水库工程简介

 

小海坝水库位于大方县城南2公里。水库集雨面积0.4平方公里，均质土坝，最大坝高13米，总库容18万立方米,干渠2条长6.8公里，设计和有效灌溉面积均为600亩。

水库设计洪水20年一遇，流量2立方米/秒,校核洪水200年一遇，流量3.6立方米/秒，库坝区均为下三迭系沙堡湾页岩，右岸开敞式溢洪道。

1957年动工，1960年完工，总投资16万元。
