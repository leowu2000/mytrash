林场水库工程简介

   林场水库位于威宁县城东部65公里，地处二塘镇新合村境内，大坝地理位置东经104°41′51″，北纬26°50′10″，属长江流域乌江支流二塘大河。该水库大坝为均质土坝，坝长85米，坝高18.3米，坝顶宽5.5米。迎水面采用块石护坡，坝坡比为1:2.5，背水面为草皮护坡坝坡比为1：2。总库容为23万立方米，集雨面积0.7平方公里。放水管为D300毫米铸铁管。溢洪道为开敞式，开挖后未衬砌海底，无消力池。

   林场水库原设计以灌溉为主，兼顾防洪。设计灌溉面积2750亩，其中田150亩，土2600亩。防洪保护面积1200亩。由于该工程大坝完工后，未进行渠系配套，故无灌溉面积效益，现仅有1200亩的防洪保护面积。在保护区内有3个村3000人，新合磷肥厂，以及大湾至贵阳铁路干线。

   该水库不能发挥灌溉效益的主要原因在于没有进行渠系配套续建。如将渠系配套完善，最终可达效益为灌溉2750亩，其中田150亩，土2600亩。
