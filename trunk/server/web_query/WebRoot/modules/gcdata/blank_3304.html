板桥水库简介

板桥水库工程位于赫章县城南面的兴发乡丫口村，距县城52公里，水库下游（赫章县内）人口0.6万人和承担下游11000亩的防洪保护面积，灌溉3500亩(经复核)，是一座集灌溉、防洪、发电等功能于一体的综合性水库。板桥水库设计标准采用50年一遇洪水设计，1000年一遇洪水校核。库区溢洪道下泄量466ｍ3/Ｓ；当超过50年一遇洪水时，就可能对下游造成洪灾。板桥水库为料土斜墙大坝，通过大坝除险加固后现最大坝高为36.5ｍ，坝顶高程1895.8ｍ，坝顶长180.0ｍ，坝顶宽6.0ｍ，大坝上游坡比为1：2.68，下游坡比为1：2.5，溢洪道为左岸坡式溢流，布置在大坝左侧的山体间，溢洪道底高程1889.5米，溢流堰顶高程1890.7米，溢流前缘净宽50米，中部净宽30米。通过除险加固后，坝体无渗漏和其它异常情况.

板桥水库所处区域地震基本烈度小于VI度，区域构造无复活现象，水库无诱发地震，区域稳定性好。库区无大的构造通过，无岩溶和临谷渗漏，库岸条件好。库水水源含泥沙量大。原坝体渗漏的主要部位为坝体、右岸坡。对坝体进行防渗灌浆和铺设土工膜右。对老放水涵洞进行封堵，新开放水涵洞，溢洪道加宽加固处理，现在已基本是比较安全的。
