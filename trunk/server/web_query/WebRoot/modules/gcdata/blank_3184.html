螺丝塘水库工程简介

 

螺丝塘水库位于大方县城北1公里。水库集雨面积0.5平方公里，均质土坝，最大坝高5.5米，总库容18万立方米,干渠1条长3.1公里，设计和有效灌溉面积均为600亩。

水库设计洪水20年一遇，流量2.08立方米/秒,校核洪水200年一遇，流量4.7立方米/秒，左岸开敞式溢洪道，坝址为下三迭系沙堡湾页岩。

1957年动工，1959年建成，总投资15万元。
