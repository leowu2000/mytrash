文家桥水库工程简介

 

文家桥水库(原名红卫水库)位于金沙县东南36公里的安底镇后寨村。水库集雨面积14.6平方公里，坝型为浆砌石三心单曲拱坝，最大坝高32.3米，总库容312万立方米，设计灌溉面积14200亩(包括结瓜水库川洞水库的灌溉面积3000亩在内)，干渠4条长35公里，支渠15条长56公里,解决20000人和5000头牲畜的人畜饮水。

水库设计洪水30年一遇，流量57.2立方米/秒；校核洪水200年一遇，流量83.0立方米/秒，坝址为中上寒系娄山关白云质灰岩，坝顶溢流，溢流段长度21.4米，低鼻坎挑流消能。

1974年动工，1977年建成。1982年至1991年先后5次进行渠系配套，防渗衬砌，增建人畜饮水工程，累计总投资330.8万元。文家桥水库为安底镇的农业发展和村镇供水发挥了重要作用，是毕节地区灌溉效益发挥较好的水库之一。
