过狮河水库工程简介

 

过狮河水库位于纳雍县城以东14公里的王家寨镇。坝型为浆砌石双曲拱坝，最大坝高35.1米，总库容320万立方米，设计灌溉面积9020亩，有效灌溉面积达3680亩。干渠2条长25公里，支渠4条长5.8公里,。坝后电站装机容量2×40千瓦。

水库设计洪水50年一遇，流量70立方米/秒；校核洪水300年一遇，流量105立方米/秒，坝顶溢流，溢流段长度15米，高鼻坎挑流消能。坝址地层为二迭系峨眉山玄武岩，“V”形河谷，两岸基本对称，在两坝肩1612.2米高程以上因岩石风化破碎，增设重力礅。

1978年动工，1980年建成。1983年至1985年续建完成至现有规模，累计总投资321.5万元。
