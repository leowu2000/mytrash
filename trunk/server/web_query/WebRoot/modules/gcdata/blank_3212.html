工程简介

黔西县落底田水库属小(二)型灌溉工程，水库位于黔西以西21公里的锦星乡红湖村，东面傍公鸡岭、西面临凹水河约1.5km，其地理坐标为东经105○48′、北纬27○00′，有村级公路直通坝上。水库坝址以上集水面积0.48km2，属乌江水系。

水库流域内大多为石灰岩类地层，岩溶发育，库区内的垦植不多，但天然植被较好，水土流失较为轻微。该水库下游大多为丘陵田坝，坝址以上无河流汇入，上、下游亦无其它水利工程。
