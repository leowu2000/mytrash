官田水库工程简介

 

官田水库属长石镇政府管理的小<一>型水库工程，位于大方县北部的长石镇境内巨石村,地理座标东经105°42′18″,北纬27°34′03″,距县城57km,距长石镇街上6km，现大纳公路从坝顶穿过,交通较为便利。该工程于1957年由原长石区委组织修建,1959年大坝基本完工,1966年完成渠系配套；从1976年～1991年不断对大坝、溢洪道、渠道进行加固和防渗处理,现大坝运行状况良好。

水库海拔高程在1350m左右,属赤水河水系二道河流域。坝址以上集雨12.24km2,水库植被较好,岸坡植被达62%,入库基流0.05m3/s,区域为黔西高原过渡带的中山缓坡丘陵谷地和洼地,出露地层主要有二迭系栖霞矛口灰岩,寒武系娄山关群白云岩,奥陶系湄潭组等,岩溶发育；土壤多为黄壤土。

整个库区地层主要为寒武系娄山关群结晶白云岩、泥质灰岩、泥页岩；大坝绝大部份坐落在湄潭组的沙、页、泥岩地层上,岩层倾向上游偏左,倾角10o左右,大坝为均质土坝,从地质角度来说,是完全适应坝基要求的,而且隔水性能良好。

官田水库现有均质土坝一座,最大坝高24m,坝顶长131m,顶宽10m,水库总库容323.5万m3,兴利库容260万m3,大坝坝顶高程1359.00m,溢洪道堰顶高程为1355.37m,设计为50年一遇,校核为300年一遇；当校核洪水标准在300年一遇时,最大溢洪水深1.66m,风浪爬高0.59m,大坝尚有1.38m的安全超高。

该库主要以灌溉为主,兼顾长石镇人畜饮水,现有渠道17.7km,其中：右干渠13.6km,中干渠4.1km;设计灌溉面积6000亩,实际灌溉面积2200亩,兼顾约5000人/1200头蓄用水。

水库溢洪道设于左坝肩,为开敝式,底宽20m,长153m,最大泄洪量63.3m3/s。

该库上游无拦水、蓄水等水利设施和厂矿,原生植被较好。下游有一大溶洞,洞侧是石坪街(巨石村驻地),穿洞后有锅圈等村组。
