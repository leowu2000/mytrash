杨家湾水库简介

    杨湾桥水库位于威宁县城西部，属长江流域，金沙江洛泽河支流，白水河发源地，距县城14公里，坝址控制流域面积97.35平方公里，工程于一九五八年九月十五日动工，一九五九年六月十八日大坝完工，一九八七年九月对大坝除险，加固。配套、于一九九零年十月竣工。工程由省水利水电勘测设计院设计，县水利施工，主管单位为威宁县水利局。

    大坝为均质土坝，最大坝高13.6米，坝顶长375米，坝顶高程2191.8米，总库容0.3622亿立方米。水库存以防洪、供水养鱼等综合效益，目前主要供县城生活用水，防洪保护范围小海镇，羊街镇、县洒精厂，38000亩土地，16个村委员21000人，洪水设计标准为百年一遇设计，1000年一遇校核，泄洪能力达到设计标准。
