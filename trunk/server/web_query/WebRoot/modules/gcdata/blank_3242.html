工程简介

    晏家沟水库位于金沙县沙土镇群立村、距沙土镇约4km，属乌江流域偏岩河上花滩河上游的一小支流。地理位置东经106°35′，北纬27°23′，有一条乡村公路通过。水库集雨面积1.38km2，水库总库容15.59万m3，防洪库容2.83万m3，属小二型水库，是一座以灌溉为主的水利枢纽工程。设计灌溉面积500亩，有效灌溉面积400亩。

    水库工程枢纽由大坝、溢洪道、放水系统等组成。水库总库容15.59万m3，属小二型水库工程，洪水标准按20年一遇设计，100年一遇校核。该水库大坝类型为浆砌石拱坝，坝顶高程841.767m，坝长98.8m，坝顶宽1.5m，最大坝高14.92m。溢洪道为坝身实用堰。放水设施为∮300mm闸阀。
