田冲水库工程简介

 

田冲水库位于黔西县以西15公里的林泉镇清毕公路左侧，水库集雨面积2.9平方公里，均质土坝，最大坝高15.8米，总库容19立方米,设计灌溉面积800亩,有效灌溉面积650亩,干渠1条长1.8公里，支渠2条长0.2公里。

水库设计洪水20年一遇，流量3.3立方米/秒,校核洪水200年一遇，流量4.9立方米/秒，左岸开敞式溢洪道，因水库上游有马场坝天然凹地调蓄洪水，故洪量较小。坝址为三迭系白云质灰岩、页岩。

1957年动工，1972年建成。总投资13.2万元。
