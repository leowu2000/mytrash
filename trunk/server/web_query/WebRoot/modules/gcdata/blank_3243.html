西堰坎水库工程简介

 

西堰坎水库位于金沙县城东北12公里的岩孔镇光明村，为箐河水库南干渠尾水结瓜水库。水库集雨面积0.2平方公里，均质土坝，最大坝高22.5米，总库容70万立方米，设计和有效灌溉面积3000亩，干渠1条长3.4公里，支渠1条长1.3公里。

水库设计洪水200年一遇，流量4.8立方米/秒；无溢洪道，坝址为土基。

1978年动工，1980年建成，总投资30.4万元。
