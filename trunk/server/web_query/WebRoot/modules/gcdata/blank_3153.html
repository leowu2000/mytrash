小果老水库工程简介

 

小果老水库位于毕节县东北35公里的阿市乡，水库集雨面积24平方公里，均质土坝，最大坝高15.5米，总库容23万立方米，干渠1条长9公里，设计灌溉面积1000亩,有效灌溉面积100亩。

水库校核洪水20年一遇，流量17立方米/秒，校核洪水200年一遇，流量32.3立方米/秒，利用左岸天然垭口泄洪。

工程自1973年动工，1974年建成。总投资16.1万元。
