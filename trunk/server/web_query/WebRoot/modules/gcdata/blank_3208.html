鹅项井水库工程简介

 

鹅项井水库位于黔西县以西18公里的锦星乡。水库集雨面积1.8平方公里，均质土坝，最大坝高15.8米，总库容64万立方米，设计灌溉面积1200亩，有效灌溉面积1000亩。干渠1条长6.5公里。

水库设计洪水10年一遇，流量13.8立方米/秒；校核洪水40年一遇，流量19.7立方米/秒，左岸开敞式溢洪道泄洪，坝址为土基。

1974年动工，1976年建成，国家补助投资57.0万元。

该水库效益显著，但坝身有些漏水，设计及校核洪水标准不够，八十年代末期，曾对坝身作塑膜防渗，溢洪水深及坝顶加高，得以正常运行。
