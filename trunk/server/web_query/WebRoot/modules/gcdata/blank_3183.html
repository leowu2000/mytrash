高店大水沟

    高店大水沟属高店乡管理的小<二>型水库工程，位于大方县城西南,距城28km,大方-白布公路从坝右侧半山通过,交通十分还算便利。该工程于1966年由县农林科水电组进行设计，原城关区委组织白布公社群众修建,工程于1966年11月动工,1976年3月完成主体工程,建有大坝1座。水库大坝建成后,国家又数次拔款进行续建配套,从1987～1992年起进行过多次改造,1992年完成渠系建筑物改造；该库以农灌为主。
