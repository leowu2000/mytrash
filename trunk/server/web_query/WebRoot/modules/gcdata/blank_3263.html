红旗冲水库工程简介

 

红旗水库位于织金县西北33公里的金龙乡细木村。水库集雨面积3.7平方公里，均质土坝，解决1000人，600头牲畜的人畜饮水。最大坝高20.3米，总库容52万立方米，设计灌溉面积3000亩,有效灌溉面积478亩，主要原因是渠系尚不配套，放水设备漏水，水库不能蓄满。

水库设计洪水20年一遇，流量26立方米/秒；校核洪水200年一遇，流量39立方米/秒，左岸开敞式溢洪道泄洪，坝址为土基。

1958年动工，1977年建成，1990年～1991年对坝体及坝基进行灌浆处理，大坝迎水坡铺塑料膜防渗，总投资38万元。
