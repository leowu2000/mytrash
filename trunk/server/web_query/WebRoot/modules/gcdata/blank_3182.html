大水沟水库工程简介

 

大水沟水库(城关)位于大方县城南3公里的清毕公路旁，水库集雨面积1.1平方公里，均质土坝，最大坝高20米，总库容18万立方米,干渠2条长5.9公里，设计灌溉面积1500亩,有效灌溉面积1500亩。

水库设计洪水20年一遇，流量5.5立方米/秒,校核洪水200年一遇，流量9.7立方米/秒，右岸开敞式溢洪道，坝址及库区均为下三叠系沙堡湾页岩。

1957年动工，1959年建成，1987年对大坝进行加固和塑膜防渗处理。累计工程总投资23.8万元。
