移山水库工程简介

 

移山水库位于大方县东南面35公里的黄泥塘镇在拱村。水库集雨面积15.3平方公里，坝型为圬工重力坝，最大坝高36.5米，总库容520万立方米，干渠1条长36.4公里，支渠17条长88公里，设计灌溉面积11000亩，实灌面积到达9355亩，坝后电站装机1×100千瓦，解决1000人，500头大牲畜饮水困难。

水库设计洪水为50年一遇，流量98.7立方米/秒；校核洪水200年一遇，流量131立方米/秒，左岸开敞式溢洪道泄洪，坝址为三叠系九级滩泥质灰岩。

工程于1965年动工，1975年建成，1977年至1993年间，又多次进行渠系配套和防渗衬砌，累计工程总投资519.9万元
