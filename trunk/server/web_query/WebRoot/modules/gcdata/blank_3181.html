大哑口水库

    大哑口水库属小屯乡人民政府管理的小<二>型水库工程，位于大方县南部的小屯乡滑石村境内,距县城8km,贵毕高速公路从水库右坝肩30m处通过，交通较为便利。该工程于1991年由大方县水电局勘测设计队设计上报,资金下达后,大方县水电局和小屯乡人民政府共同组建工程指挥部，工程于1992年11月开工建设,1994年4月大坝工程完成，属小屯乡人饮专用水库。
