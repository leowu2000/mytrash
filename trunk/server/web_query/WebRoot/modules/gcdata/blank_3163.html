工程概况

东郊水库属城关国营水管所管理的小<二>型水库工程，位于大方县城东郊,距城1km ,专用公路可直达坝顶，交通十分便利。该工程1997年列为大方县大方镇6村渴望工程规划,同年底由大方县水电局设计上报，1998年初地区水电局审查通过并下达资金157.76万元；1998年2月县人民政府从水电、大方镇、财政、计经、国土、建设、自来水公司和城关水管所抽人组建“工程指挥部”，3月完成土地征用等前期工作，同年4月2日动工,2000年7月工程竣工, 因地质因素,竣工后发现库区渗漏,并绕左坝肩在反滤层左岸形成集中渗漏通道,邀请上级有关地质部门论证处理均未收到效果，为确保水库下游县城南门大沟一带群众安全，从2001年起县防汛办对东郊水库下达空库度汛计划,汛期或枯水期严禁关闭放水闸门,仅保留死水位以下运行。
