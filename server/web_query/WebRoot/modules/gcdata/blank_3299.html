结里水库简介

    结里水库地处威宁县城东南部炉山镇新装村境内，距县城36公里。大坝地理位置东经104°31′39″，北纬26°49′42″。属长江流域乌江支流二塘河上游。

    结里水库为均质土坝，坝高13.08米，坝长120米，坝顶宽3米，迎水面为块石护坡，坝坡比为1：2。背水面为块石护坡，坡比为1:1.5，总库容21.4万立方米，集雨面积2.2平方公里。该工程完工时原结里乡政府明确3人进行管理，但由于运行多年来存在着配套不完善，管理松懈等问题，现溢洪道进口宽16米，均宽8米，长100米。结里水库自1976年5月建成后，曾进行过三次除险加固处理，均由威宁县水电局设计，原结里乡组织施工。结里水库是以灌溉为主，兼顾防洪养鱼的小（二）型水库，原设计灌溉面积3000亩，其中田1000亩，防洪保护面积3000亩，现在实际灌溉面积950亩，其中田250亩，灌区水稻田每亩单产500斤，总产11.5万斤，旱地每亩单产700斤，总产49万斤，现在防洪保护面积1500亩，同时，保护范围内，使新装村1500人免受洪水威胁。

    该水库如将渠系及建筑物修复完善并加强管理，最终可达到灌溉面积3000亩，其中田1000亩，防洪保护面积3000亩的设计要求。另外，该水库现有养鱼面积40.2亩，年产成鱼0.45万斤。
