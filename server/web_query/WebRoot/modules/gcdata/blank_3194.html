工程简介

         附廓水库位于黔西县城西北l7Km处的洪水乡皮家河上游。属于长江流域乌江水系，是乌江一级支流——野济河的干流，于箐口乡荆江村汇入乌江上游鸭池河。
      水库发源于金坡乡的老沟头，干流雨灌河在观音岩处变为伏流，于水淹坝出现。另一支流发源于河沟头，在沙 莫莫处变为伏流，于奶母田处出露，向东流约400m后再次转入地下，于水淹坝出流，与雨灌河汇合后，又转入地下，在三岔土处出流后又成为暗河，最后在水库库尾鱼洞出露。水库流域内碳酸盐类岩石分布广泛，岩溶相当发育，岩溶地貌如溶丘、洼地、峰丛、漏斗、伏流等分布普遍。水库坝址以上集水面积72KKm2，主河道长14 Km，主河道平均坡降20．2‰。
       附廓水库流域属于中亚热带向北亚热带过渡的季风气候区，夏无酷暑，冬无严寒。根据黔西气象站历年资料统计，多年平均气温13.9℃，极端最低气温一10.4℃，极端最高气温35.4℃。多年平均日照时数l284．7h，年平均无霜期271．3天，多年平均水面蒸发量l288.4.mm，多年平均风速l.6m／s，全年以NE风居多，多年平均降水量984．1 mm，降水量年内分配主要集中在4一9月，占全年的81.8％。灾害天气有春旱、倒春寒、冰雹、秋绵雨、暴雨等。
