烂田坝水库

    烂田坝水库位于金沙县城西北面，距县城62公里，地处太平乡附阳村的烂田坝，属长江水系赤水河流域水边河支流。地理位置东经105°54′，北纬27°34′，该水库属土坝，坝顶长77.9米，坝顶宽3.5米，总库容18.8万立方米，水库溢洪道采用河岸开敞式溢洪，底槛高程1349.3米，底宽8.4米，最大水深2.1米，最大泄洪量40.7m3/s，长度30.0米，没有公路与外界相通。

    水库工程枢纽由大坝、溢洪道、放水系统等组成。水库总库容18.8万m3，属小二型水库工程，洪水标准按20年一遇设计，100年一遇校核。该水库大坝类型为土坝，坝顶高程1352.6m，坝长77.9m，坝顶宽3.5m，最大坝高12m。溢洪道为河岸开敞式。放水设施为放水涵管。
