小箐沟水库

    小箐沟水库属在建的小<一>型水库工程，位于大方县东南部的羊场镇陇公村境内,距县城5km,老清-毕公路3km处(黄河谷)有水库专用公路直达水库大坝，交通较为便利。该工程于1999年11月进入可研,2000年4月贵州省国际工程咨询公司对项目进行实地考察评估,同年8月由毕节地区勘测设计院设计上报,2001年11月开工建设,承建单位为贵州省水利厅机械化施工队,2003年12月大坝工程完成，现还未进行工程验收；水库由大坝、提水站、10kv输电线路、进厂公路组成,工程总投资1315.12万元；属大方县城人饮专用水库，现水库己开始蓄水使用，从2004年起自来水公司派员驻站抽水，2007年8月6日水库溢洪,因工程未验收,现还未落实具体管理单位；按属地管理原则,防洪责任由羊场镇人民政府承担。
