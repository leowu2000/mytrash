乐溪水库工程简介

 

乐溪水库位于威宁县以东30公里的炉山镇乐溪村。水库集雨面积11平方公里，均质土坝，最大坝高14米，总库容84万立方米，设计灌溉面积5000亩，有效灌溉面积400亩，干渠1条长0.8公里，引水式电站装机容量2×250千瓦，解决1800人，2800头牲畜饮水困难。

水库校核洪水300年一遇，流量67.3立方米/秒，右岸开敞式溢洪道。

1974年动工，1976年建成，1986年电站投产，并对溢洪道作了扩建，累计总投资35.3万元(不含电站)。
