千斤水库工程简介

 

千斤水库位于织金县西北35公里的金龙乡关心村。水库集雨面积1.1平方公里，均质土坝，最大坝高8.5米，总库容23万立方米，灌溉面积436亩，干渠2条长4.5公里，支渠1条长1.1公里。

水库设计洪水20年一遇，流量17.2立方米/秒，校核洪水200年一遇，流量23.8立方米/秒，利用左岸开敞式溢洪道泄洪，坝址为土基。

1957年动工，1958年建成。1981年至1982年对大坝进行防渗处理和溢洪道改建。总投资36万元。
