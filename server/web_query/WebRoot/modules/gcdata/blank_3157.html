野鸡水库工程简介

 

野鸡水库位于毕节县西南55公里的之阴底乡野鸡村，水库集雨面积2.1平方公里，坝型为粘土斜墙加柔粘防渗，最大坝高14.0米，总库容19立方米,设计灌溉面积3000   亩其中水浇地2500亩，干渠1条长1.9公里，输水渠后埋设塑管长2公里。

水库设计洪水频率20年一遇，流量15.5立方米/秒,左岸开敞式溢洪道。

水库始于1956年冬，1959年底建成，坝高10米土坝一座，1979年至1991年经过4次防渗固处理和改建，始建成现有规模，累计工程总投资48.9万元。
