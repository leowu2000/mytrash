凉水井水库工程简介

 

凉水井水库位于大方县北面60公里的八堡乡复兴村。水库集雨面积1.2平方公里，粘土心墙土坝，最大坝高18米，总库容47万立方米，干渠3条长2.5公里，支渠3条长2.7公里。设计灌溉面积2000亩,有效灌溉面积1280亩。

水库设计洪水20年一遇，流量5.2立方米/秒；校核洪水200年一遇，流量9.3立方米/秒，溢洪道设在库尾山垭口，坝址为页岩。

1965年动工，1973年建成，总投资17.2万元。
