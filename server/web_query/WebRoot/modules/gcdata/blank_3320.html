  香椿树水库简介

    赫章县香椿树水库位于赫章县城东面的城关镇城关片区的青山村香椿树组，距县城直线距离8ｋｍ，公路30ｋｍ，地理坐标：东径104046＇32″，北纬27006＇21″。属乌江水系七星河支流上游。水库坝址位于一半山区溪沟地带，库区边上属于青山村村办林场，植被比较好，水土流失很轻微，因此该水库到现在为止还没有较大的淤积，坝址以上集雨面积1.4㎞2，该水库总库容29万ｍ3，属小（二）型水库。工程的兴建当时主要是为了解决下游1200亩田地的灌溉（当时设计灌溉面积），是一座设计以灌溉为主要功能的水库。但工程建成后，渠道不配套，到现在为止也没有真正发挥灌溉效率，自2001年开始，赫章县自来水公司才把其作为县城饮用水源，才真正有效益，但现在这水库还成了对下游的人民生命财产和土地等具有较大防洪责任的水库，主要任务是承担着保护下游青山村香椿树组、卸旗村大沟边组、黄泥村下部等4500人的生命财产、550亩土地、大沟边一个水厂、一所学校、养护公区及326线近500米公路等的防洪保护。
