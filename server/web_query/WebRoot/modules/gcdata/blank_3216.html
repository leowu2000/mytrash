木弄水库

    木弄水库位于黔西县协和乡木弄村，距协和乡所在地3公里。其地理坐标是东经105○16′北纬27○02′。流域内无河流，在北面协和乡与中坪镇交界的榨仡河属乌江水系，往东流入二海处汇入乌江。库区平坦，中下部属九级滩页岩，有较好的隔水层。库区来水全由大气降水补给，排水形式主要通过地表裂隙、孔隙及洼地、漏斗、溶洞渗入地下。坝址以上集雨面积2.79km2，水库总库容71.8万m3，其中兴利库容51.50万m3，死库容8.5万m3，灌溉面积2500亩。

木弄水库始建于1974年11月，于1978年11月竣工，为小（二）型水库，是协和乡境内唯一的水库。

    该水库距协和乡3公里，地理坐标为东经105○16北纬27○02′。水库集雨面积为2.79平方公里，呈扇形，植被一般。实测总库容71.8万立方米，兴利库容51.05万立方米。现建有均质土坝一座，大坝建于木弄槽出口处。坝高16m，坝长120米，坝顶宽4米，大坝迎水面坡为1:3.1，背水面坡度1:2.5。溢洪道在右坝肩，距坝下游右端150米处有一消水洞，作为溢洪道排水之用。渠道经890米隧洞进入跃进大塘，沿海坝、杨柳、花苗寨、青木等六村，渠道总长5.42公里，设计灌面2500亩。木弄水库上下游均无重大水利工程设施。
