  工程简介

     赫章县白草坝水库位于赫章县城东面的平山乡平山片区的农庄村中沟组的大海子处，距离县城50ｋｍ，地理坐标：东径104052‘32″，北纬27011’21″。属乌江水系七星河支流上游。水库坝址位于一半山区溪沟地带，距326国16公里，库区边上属于平山国有林场，植被比较好，水土流失很轻微，因此该水库到现在为止还没有较大的淤积，坝址以上集雨面积0.8㎞2。
该水库总库容23万ｍ3，属小（二）型水库。工程的兴建当时主要是为了解决下游500亩稻田的灌溉（设计灌溉面积），是一座以灌溉为主要功能的水库。
     白草坝水库设计标准采用20年一遇洪水设计，200年一遇洪水校核，水库下泄流量为4.8ｍ3/Ｓ；当超过20年一遇洪水时，对下游也造成不了多大灾害。白草坝水库为均质土坝，最大坝高为18.2ｍ，坝顶高程1720.00ｍ，坝顶长210.00ｍ，坝顶宽4.0ｍ，大坝上游坡比为1：2.0，下游坡比为1：2.0，溢洪道为左岸坡式溢流，其布置在大坝左侧的山体间，溢洪道底板高程1718.42米，溢洪道进口净宽4米。
