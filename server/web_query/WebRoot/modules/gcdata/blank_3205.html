烂田冲水库工程简介

 

烂田冲水库位于黔西县东北35公里的重新镇。水库集雨面积1.7平方公里，均质土坝，最大坝高14米，总库容36万立方米，设计灌溉面积1213亩,有效灌溉面积1041亩。干渠1条长5.3公里。

水库设计洪水20年一遇，流量19.5立方米/秒；校核洪水200年一遇，流量29.3立方米/秒，右岸开敞式溢洪道。坝址为九级滩页岩、泥质灰岩。

1963年动工，1964年建成，总投资9.3万元。
