箐河水库工程简介

 

箐河水库位于金沙县西北30公里的桂花乡鹦歌桥。水库集雨面积8.9平方公里，均质土坝，最大坝高21.0米，总库容324万立方米，设计灌溉面积11378亩(包括结瓜水库西堰水库灌溉面积3000亩在内)，干渠1条长23.5公里，支渠2条长7公里。

水库设计洪水50年一遇，流量60立方米/秒；校核洪水300年一遇，流量86.3立方米/秒，右岸开敞式溢洪道泄洪，坝址为寒武系白云质灰岩。

1965年动工，1966年建成。1967年至1991年先后4次进行渠系及建筑物配套防渗，大坝全面加固防渗处理，累计总投资309.8万元。是毕节地区灌溉效益发挥效好的水库之一。
