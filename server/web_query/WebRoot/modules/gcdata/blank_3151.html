崔家沟水库工程简介

 

崔家沟水库位于毕节县东北面74公里的清水铺镇崔家沟。水库集雨面积1.4平方公里，均质土坝，最大坝高22.5米，总库容35万立方米，干渠1条长1.4米，设计灌溉面积2770亩，其中水浇地1350亩，解决4600人、3500头牲畜的人畜饮水困难。

水库设计洪水20年一遇，流量11.8立方米/秒；校核洪水200年一遇，流量22.4立方米/秒，右岸开敞式溢洪道泄洪。库区及坝址均为页岩。

工程始建于1973年，1975年大坝建成，1990年至1993年对大坝进行灌浆防渗加固和渠系配套，右坝肩灌浆处理，迎水面增设柔粘防渗，累计工程总投资135.5万元。
