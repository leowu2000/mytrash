水淹水库工程简介

 

   水淹水库位于黔西县以南22公里的金碧乡水淹坝。水库集雨面积15.3平方公里，主坝为均质土坝，最大坝高19.7米，总库容305万立方米，干渠2条长31.4公里，支渠6条长24.3公里,设计灌溉面积8650亩,有效灌溉面积6500亩。

   水库设计洪水30年一遇，校核洪水300年一遇，该水库为一岩常人洼地，下游无河道泄洪，采用蓄洪方式，防洪库容128.6万立方米。在付坝中设浆砌石溢流坝以宣泄非常洪水。坝址为下三迭系茅草铺灰岩夹页岩层。

   工程1966年动工，1969年建成。1977年至1988年曾三次对渠系进行配套、防渗、加固。累计总投资196.2万元
