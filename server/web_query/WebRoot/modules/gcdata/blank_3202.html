龙家槽水库工程简介

 

龙家槽水库位于黔西县以西15公里的林泉镇。水库集雨面积0.9平方公里，均质土坝，最大坝高7.9米，总库容24万立方米，设计灌溉面积800亩,有效灌溉面积400亩，干渠1条长2公里。

水库校核洪水200年一遇，流量29.2立方米/秒，左岸开敞式溢洪道。

1975年动工，1976年建成，国家补助投资2.4万元。

八十年代未，利用龙家槽的库水，配以输水管道、蓄水池、使锦星坝子实现了烤烟、玉米制种基地水浇化。
