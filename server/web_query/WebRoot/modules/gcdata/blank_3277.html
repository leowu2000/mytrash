小坝水库工程简介

 

小坝水库位于纳雍县以南23公里的张家湾镇。水库集雨面积6.3平方公里，均质土坝，主坝最大坝高14米，副坝高12米，总库容42万立方米，设计灌溉面积2000亩,有效灌溉面积1850亩。干渠2条长2.9公里，解决2470人，1740头牲畜的人畜饮水。

水库设计洪水30年一遇，流量32立方米/秒；校核洪水200年一遇，流量52立方米/秒，在副坝右岸设开敞式溢洪道泄洪。

1957年动工，1958年水库建成，1990年至1991年对主坝迎水增设柔毡防渗和溢洪道及放水设备改建。累计总投资21.7万元。
