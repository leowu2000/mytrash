工程简介

工程概况

露海水库属核桃乡管理的重要小山塘水库工程，位于大方县城西南部的核桃乡伐木村境内,距县城25km ,距政府驻地6km,交通条件一般。该工程于1958年冬动工兴建,未进行勘测设计和审批手续,由达溪区水利站进行技术指导，公社组织群众实施；经过3年多苦战，于1962年完工, 建成均质士坝1座；水库总库容8万m3；建库以来,由于工程质量差和设施不配套等诸方因素,工程一直末能发挥效益,渠道仅通过一次水,后闸门等放水设备被盗,启闭房及管理房被拆除，水库闸门涵洞被异物卡死和泥沙填塞,故水库报废,1985年废减为山塘移交当地管理。
