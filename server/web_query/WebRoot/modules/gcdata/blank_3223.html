工程简介

粮田水库始建于1993年4月，于1994年10月竣工，为小（二）型灌溉工程。大坝出口即为灌区，灌区集中。水库集雨面积为0.77km2，实测总库容17.68万m3，兴利库容9.74万m3，死库容2.64 m3，设计灌溉面积500亩。现建有主坝一座，为均质土坝，坝高15m，坝长64m，坝顶宽3.0m，大坝迎水面坡度为1:4.0，背水面坡度为1:2.25，溢洪道建在右坝端，为正槽式宽顶堰型，全长87m,由进水段（5m）、控制段（10m）、渐变段（10m）、陡坡段（47.4m）、消能段（14.6m）组成，另外还配套了10m长的尾水渠。

粮田水库正常蓄水位1279.25m，设计洪水位1280.12m，校核洪水位1280.42m；主坝坝顶高程1281.70m。
