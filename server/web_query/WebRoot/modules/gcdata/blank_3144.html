倒天河水库工程简介

 

倒天河水库位于毕节市西北部，距市区 3km ，所在河流为六冲河一级支流白甫河上游，属长江水系；坝址以上集水面积 124km 2，其中：地表 83km 2，地下 41km 2,坝址以上主河道长 20.6km ，河道平均坡降9.3%。多年平均年径流量6840万m3，水库总库容1880万m3，属中型水库。因其位于毕节城区上游，水库承担着毕节市城区的防洪任务，根据毕节城区的防洪规划，要达到50年一遇防洪标准，必须充分利用倒天河水库的调洪削峰作用；同时，倒天河水库是毕节市城区的主要供水水源，现日供水量6万m3，远期日供水量8.7万m3；根据毕节城区总体规划，到2010年人口将达30万，需水量10万t/d；此外，水库还具有灌溉和发电等效益，灌溉面积 390.4ha (5856亩)，年发电量270万kw/h。

倒天河位于白甫河的上游，河流为山区雨源性河流，洪水主要由流域内的暴雨形成。因受到伏流、落水洞的滞蓄，产生的洪水相对较小，洪水汇集时间短，退洪时间长，一次洪水历时为2～3天，洪水过程一般呈双峰型，主峰在后，主要出现在5～8月份。

倒天河水库下游毕节市城区包括市东、市西、大新桥、观音桥、流仓桥、三板桥六个办事处，是毕节市人民政府和毕节地区行政公署所在地，是毕节地区政治、经济、文化和交通运输的中心。面积约 5.8km 2，人口18.4万人，除汉族外，还有彝、苗等十九个少数民族，是一个多民族聚居城市。全市工农业总产值59957万元，其中工业总产值43455万元，农业总产值16502万元。有主要工业、企业500余家，包括烟草、化工、建材、电力、煤炭、皮革、机械、轻纺等行业。

毕节市倒天河水库总库容1880万m3，属于中型水库，工程等别为III等，主要建筑物为3级，次要建筑物为4级。大坝为浆砌石重力坝，泄洪方式为坝顶溢流。该工程分三期建成，一期工程于1957年由原省水利局负责，建成了坝高为 13m 的浆砌石重力坝，库容100万m3。二期工程于1963年由毕节专署农田水利局进行扩大初步设计，设计加高大坝 10.5m ，1964年动工兴建，形成的水库规模为最大坝高 23.5m ，正常蓄水位 1548.30m ，库容为650万m3，校核洪水位 1552.12m ，总库容1010万m3。2001年，倒天河水库除险加固工程开始实施，该工程总投资650万元，实施内容包括对坝体、基础和两岸基岩作灌浆处理，溢洪道改造、底孔、取水口处理。工程于2002年10月完工，提高了对水库防洪和供水能力。为下一步大坝加高工程打下了良好的基础。2003年5月开始对倒天河水库进行了加高扩建，扩建后水库最大坝高 36m ，正常蓄水位 1557.30m ，相应库容1450万m3；校核水位 1559.93m ，相应总库容1880万m3。该工程完工后，将水库日供水能力由原来2.7万吨提高到7.3万吨，并将下游毕节市区防洪标准由30年一遇提高到50年一遇。
