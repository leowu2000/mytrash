工程简介

  工程简介

      马槽沟水库位于赫章县城东面25公里的野马川镇石亚村，属六冲河上游。流城面积1.1平方公 里。
      该水库为均质土坝，坝高21.5米，坝顶长78米，总库容21万立方米，其中防洪库容12万立方米。　
　    溢洪道为开敞式，其尺寸为：宽2.5米，过水深度0.2米，泄洪长20米。　
　    放水设备为坝后闸阀控制，即放水涵管为200mm钢管，采用同型号200mm闸阀控制。　
　    根据《防洪标准》，该水库应为小（二）型，工程等级为5等，主要建筑为4级，次要建筑物为5级。　
    　该水库1985年曾进行过“三查三定”，经查，该工程于1962年秋季动工兴建，1968年正式完工，1989年进行洪道防渗，并投入使用。
