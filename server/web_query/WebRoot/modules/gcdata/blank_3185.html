云龙山水库工程简介

 

云龙山水库位于大方县城东北4公里的云龙山中。水库集雨面积0.7平方公里，坝型为混凝土双曲拱坝，最大坝高27.3米，混凝土垫座高6米，总库容14万立方米。该水库主要人工湖县水泥厂、复烤厂、蚕丝厂的生产、生活用水。

水库设计洪水20年一遇，流量3.5立方米/秒；校核洪水200年一遇，流量62立方米/秒。坝顶溢流，溢流段长16米，坝址为下三迭系沙堡湾泥质灰岩。

1979年动工，1986年建成，1992年进行坝基帷幕灌浆和坝身漏水处理，总投资43.3万元。
