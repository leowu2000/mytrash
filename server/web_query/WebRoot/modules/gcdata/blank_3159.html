落脚河电站简介

    落脚河电站位于贵州省西北部乌江流域上游六冲河一级支流的白甫河上，工程坝址座落在毕节地区大方县双山镇境内的落脚河段，紧靠321国道，大坝在落脚河大桥上游600m处，电站在大桥上游约300m处。
    根据毕节地区水电局在1985年对白甫河流域水电开发规划的要求，整个白河干流分九级开发，虽然后来作了一些修改和变动，但整体格局未变，仍维持了原来的九个梯级开发原则。落脚河电站是白甫河干流规划的第七梯级电站，也是该河干流上唯一具有较好调节性能的电站。
    落脚河电站建成后并入电网运行，具有突出的调峰作用，加之电站装机占电网容量的比例较小，对电站保证出力要求相对不高，因此按年均发电量较多，装机年利用小时较少的原则确定电站装机规模。在修编过程中，拟定1.6、1.7、1.8、1.9、2.0、2.1万kw六个装机方案进行动能经济比较后，选定指标最优的2×104kw为电站的装机规模，其动能指标为：保证出力2830kw，多年平均发电量6960万kw·h，年利用小时数3480h。
    电站工程由大坝、溢洪道、底孔、发电隧洞和厂房等主要建筑物组成，修编初设的工程布置仍按原审查批准的下坝址下坝线开展工作。下坝线的工程地质和水文地质条件较好，为三迭系永宁镇第二段地层，大坝基础主要持力层为第二段第二层（T1yn2-2），且为两岸基本对称的“V”型峡谷，适宜修建拱坝。
    在下坝址下坝线建MgO砼双曲拱坝，最大坝高81m；在主河槽溢流坝段设露顶式弧形闸门控制的坝顶溢洪道；在靠发电取水口的右岸挡水坝段设冲沙兼放空底孔；发电引水隧洞布置在右岸山体内；厂房布置在坝址右岸下游300m处地形开敞平坦的冲积漫滩上，升压站设置在厂内，主变压场靠厂房布置，交通很方便。
    落脚河电站水为总库容4650万m3，装机2×104kw，属中型水库小（一）型电站，工程等级为三等，大坝为3级建设物，厂房等为4级建设物。
