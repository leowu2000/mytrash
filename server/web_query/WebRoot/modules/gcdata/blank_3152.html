雅木水库工程简介

 

   雅木水库位于毕节县东北118公里的大屯乡雅木村。水库集雨面积3.1平方公里，均质土坝，最大坝高25.8米，总库容47万立方米，干渠2条长6.8公里，设计灌溉面积3370亩,有效灌溉面积500亩。

   水库设计洪水20年一遇，流量20.3立方米/秒；校核洪水200年一遇，流量38.0立方米/秒，左岸开敞式溢洪道泄洪。

     1974年动工，1980年建成坝高23米后停建，1990后续建完成。累计工程总投资64.4万元。
