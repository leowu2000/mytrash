利民水库工程简介

 

利民水库位于毕节县城以北15公里的何官屯镇大鱼洞村，水库集雨面积19.8平方公里，坝型为桨砌石单曲拱坝，最大坝高45米，总库容746万立方米，设计灌溉面积15400亩，坝后电站装机2×75千瓦。

水库设计洪水50年一遇，流量57.4立方米/秒；校核洪水300年一遇，流量77.6立方米/秒，坝顶溢流，溢流段长度24.9米。

坝址地层为三叠系九级滩泥质石灰岩。“V”形河谷，两岸基本对称。

水库工程于1974年动工，1980年建成蓄水，坝后电站开始发电，由于设计灌溉面各90%为土变田，又大部集中在干渠37公里后的尾部，土变田效益不显著，灌溉效益长期不能发挥。1987年起将水库改为向毕节县城及何官屯镇供水，新建无压输水隧洞一座长1527米，钢筋混凝土予应力输水管一条长9984米，于1991年建成，日供水量11000吨，全部工程累计投资615万元。
