雪山水库工程简介

 

雪山水库位于威宁县东北30公里的雪山镇。水库集雨面积6.6平方公里，均质土坝，最大坝高18.6米，总库容240万立方米，设计灌溉面积9000亩，其中“三湿”下浇地7000亩，现到达有效灌溉面积900亩，引水式电站装机50千瓦，解决22000人，29000头牲畜的人畜饮水困难。干渠2条长20公里，支渠2条长27公里。

水库设计洪水20年一遇，流量38.7立方米/秒；校核洪水300年一遇，流量71.6立方米/秒，左岸无压隧洞泄洪。

1958年动工，1979年建成，1981年至1983年进行全面配套，累计总投资182万元。
