大山水库简介

大山水库位于威宁县城西南部麻乍乡戛利村境内，距威宁县城50公里处，距麻乍乡政府所在地20公里，大坝地理位置，东经104°07′00″，北纬26°43′38″，水库集雨面积0.8平方公里，坝体为均质土坝，坝高12.4米，坝长64米，坝顶宽5米，总库容8.2万立方米，放水涵管为D300铸铁管。属珠江流域北盘江支流夸都河上游。

大山水库原设计灌溉面积为2000亩，防洪保护面积500亩。由于水库大坝完工后无渠系配套，所以无灌溉效益。现水库仅对下游250亩旱地及一村民组800人起到防洪保护作用，如将水库渠系配套完善，可解决1200亩水浇地的灌溉用水和解决1500人、1600头畜的人畜饮水困难。

该水库于77年修建了大山二级电站装机一台50千瓦，78年修建了大山一级电站，装机一台50千瓦。两座电站系由乡政府、大队和生产队自筹资金进行修建，当时未考虑水源是否能够满足发电供水要求。建成后的大坝一、二级电站由于水库供水不足，不能正常发挥其效益，虽架设有25公里输电线路，解决戛利乡政府和附近3个村1800人的照明，但年发电量仅5万度，随着80年土地承包到户后，管理跟不上造成机组及线路损坏，两座电站相继停机直至报废。
