工程概况

宋家沟水库属建设局下辖大方县自来水公司管理的小<二>型水库工程，位于大方县东部的羊场镇陇公村境内,距县城6.5km,大方县城有水库专用公路直达水库大坝，交通较为便利。该工程于1985年由大方县水电局勘测设计队设计上报,1986年7月开工建设, 1988年12月大坝工程完成，1990年7月15日通过验收；水库由大坝、提水站、10kv输电线路、隧洞、进厂公路、水厂组成,工程总投资416.4万元；属大方县城人饮专用水库。
