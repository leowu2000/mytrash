扇子田水库简介

  扇子田水库简介

    扇子田水库位于赫章县城东部326国道旁，隶属平山乡中山村七里沟组，距县城40km处，属小（一）型水库，主要承担着面积为3300亩稻田的灌溉任务，大坝复核洪水标准采用30年一遇（P＝3.33%）设计，500年一遇（P＝0.2%）校核要求，扇子田水库大坝为均质土坝。主坝高29.1m，坝顶长106m，顶宽6m，坝顶高程1553.50m，坝顶设有防浪墙，长107m，宽1m，高0.9m,防浪墙顶高程1554.40m；副坝高12.4m，坝顶高程1553.69m，顶长86m，顶宽6m，顶部同样设有防浪墙，长86m，宽1m，高0.9m。主坝迎水面坡比为1：2，背水面坡比为1：2.25，副坝迎水面坡比1：3，背水面坡比1:2；放水闸门位于主坝与副坝之间山体1533.00m高程处，其外段为一城门洞形的放水涵洞，放水闸伐为￠600mm的双向转动闸伐，洞内涵管道直径为￠600mm，最大放水流量为0.55m3/s。
    扇子田水库库区基本地震烈度为小于VI度，区域地质条件的稳定性较好，水库原修建完工后投入运行后，大坝存在着一定的渗漏现象，其主要原因是筑坝时坝体的填筑土质不均匀，碾压不结实等原因等所造成。主坝和副坝仅出露二迭系上统峨嵋山玄武岩（p2O）组地层及第四系坡积物层，地层为不整合接触，前者岩性为灰绿，暗绿色玄武岩，岩层层理不清楚。后者为褐色，黄褐色粘土，含较多碎石颗粒，碎石颗粒多为玄武岩风化物残留，含有较多风化颗粒，故其渗漏系数大。该工程于1992年和1998年由地区水利局水利工程建筑施工队作过大坝灌浆处理，可仍然存在渗漏现象，后来又于2004年由毕节地区新禹水利工程建筑公司对水库作过除险加固处理，目前为止，工程枢纽还未发现有任何异常现象。
