青浦塘水库

    青浦塘水库位于黔西县林泉镇，东经105°54′，北纬27°02′，距黔西县城13km，水库大坝位于黔西至大方县道公路旁，距县城较近，交通方便。

    青浦塘水库兴建于1963年，水库坝型为均质土坝，属于小（一）型灌溉型水库，设计灌溉面积5420亩，实际灌溉面积3000，亩；总库容为114.55万m3，兴利库容为106万m3，死库容为8.5万m3；最大坝高8.22m，坝长136m，坝顶宽2m，溢洪道为开敞式自流溢流，长57m，宽2.4m，为矩形；大坝迎水和背水坡面坡度都为1：2。

    该水库由天然海子改造而成，原无正规大坝，防洪效益不明显。1963年，林泉区根据设计，组织群众完成了白泥方向（即水库的西南方向）放水建筑物的施工。1974年，由林泉区组织野坝公社群众，对野坝方向的垭口做挖方开挖，经两年多的时间完成了长900m，深15m地段的开挖任务，并拱砌好。1982年，青浦塘水库配套工程指挥部筹建；1983年2月动工，历时两年多，完成了放水建筑物2座，总干渠道一条，长1.46km,干渠条,长12.88km；隧洞一座，长924m；电灌站一处。水管宿舍一栋，152.4m2。
