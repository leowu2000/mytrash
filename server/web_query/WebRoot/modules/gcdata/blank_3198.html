卷洞门水库工程简介

 

卷洞门水库位于黔西县以北20公里的仁和乡。水库集雨面积6平方公里，浆砌石单曲拱坝，最大坝高23.2米，总库容145万立方米，设计灌溉面积4000亩，有效灌溉面积120亩，解决2000人和1000头牲畜的饮水困难。支渠4条长5.5公里。

水库设计洪水30年一遇，流量85.4立方米/秒；校核洪水200年一遇，流量124.7立方米/秒，坝顶溢流，溢流段长度50米，坝址为石灰岩基础。

1957年动工，1980年建成。总投资98.6万元。

该水库坝身原用石灰砂浆砌筑，座浆不满，产生坝身渗漏和沉陷变形，不能正常蓄水，为一病库，故灌溉效益未能发挥，目前正利用农发资金采用综合措施加固。
