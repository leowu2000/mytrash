水头上水库工程概况

工程概况

水头上水库属牛场乡管理的小<二>型水库工程，位于大方县城西南面的牛场乡黑仲村境内,距县城45km ,距牛场乡政府驻地0.8km,交通条件一般。该工程于1975年12月动工兴建,未进行勘测设计和审批手续,由马场区水利站进行技术指导，牛场公社组织群众实施；经过3年苦战，于1978年11月完工, 建成均质士坝2座；水库总库容13.7万m3，兴利库容11.1万m3，死库容0.6万m3，水库正常蓄水位1420.70m，死水位1416.30m。
