冒水水库工程简介

 

冒水水库位于威宁县以东15公里的威水公路右仙的金钟冒水村。水库集雨面积15平方公里，均质土坝，最大坝高17.5米，总库容138万立方米，设计灌溉面积6000亩，有效灌溉面积2500亩，防洪保护面积1800亩，干渠1条长9公里，支渠2条长6.5公里，渠道跌水电站一座，装机容量2×100千瓦。

水库设计洪水30年一遇，流量25.8立方米/秒；校核洪水300年一遇，流量65.5立方米/秒，坝址为土基，右岸开敞式溢洪道泄洪。

1975年动工，1976年建成，1980～1982年渠系配套完成，1980～1986年曾先后5次对大坝滑坡进行处理。九十年代初整修溢洪道，累计总投资82.5万元。
