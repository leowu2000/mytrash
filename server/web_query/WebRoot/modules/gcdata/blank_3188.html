白瓦厂水库工程简介

 

白瓦厂水库位于大方县城南5公里的白瓦厂，是一个利用天然凹地无坝堵洞的蓄水工程。水库集雨面积1.2平方公里，总库容55万立方米，干渠1条长8.1公里，设计灌溉面积1500亩,有效灌溉面积1500亩。。

水库设计洪水20年一遇，流量5.9立方米/秒；校核洪水200年一遇，流量10.5立方米/秒。

1957年动工，1959年建成，工程总投资18.3万元
