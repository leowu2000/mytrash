拦河堰水库

    拦河堰水库位于金沙县安底镇西南4公里处的香林村，地名石垭口，地理位置：东经106°24´54″，北纬27°22′12″。河流水系为长江水系乌江流域偏岩河的支流。

    拦河堰水库工程枢纽由大坝、溢洪道、放水系统等组成。工程为小(二)型蓄水灌溉工程，大坝为浆砌石单曲拱坝一座，最大坝高23.08米，坝顶宽1.1米，坝顶弧长56.1米，坝顶高程1111.56米，坝址以上集雨面积0.28平方公里，总库容21.07万方，其中：兴利库容17.92万m3，防洪库容2.62万m3，死库容1.4万m3，溢洪方式为坝顶溢流，溢洪道底槛高程1110.86米，溢洪道断面尺寸长1.1米，宽8.8米，只能符合安全超高标准，无泄洪能力，放水设备为ø200mm闸阀，最大过水流量0.724m3/s。共建渠道3.97公里，设计灌溉400亩。
