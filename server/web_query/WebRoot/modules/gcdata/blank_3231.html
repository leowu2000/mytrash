木蓑衣水库工程简介

 

木蓑衣水库位于金沙县东南45公里的长坝乡树寨村。水库集雨面积6.7平方公里，浆砌石单曲拱坝，最大坝高30.0米，总库容176万立方米，设计和有效灌溉面积均达8380亩，干渠2条长15.8公里，支渠9条长25公里。

水库设计洪水30年一遇，流量52.1立方米/秒；校核洪水200年一遇，流量75.0立方米/秒，坝顶溢流，溢流段长度20米，高鼻坎挑流消能。坝址为下三迭系玉龙山灰岩，左坝肩岩石十分破碎，故设重力礅。

1977年动工，1980年建成，1981年至1982年曾两次对右坝肩漏水通道进行灌浆处理，有效的堵住了漏水通道。1990年至1993年对渠系进行防渗、配套和建筑物改建。累计工程总投资315.7万元。
