工农水库简介

工农水库位于威宁县城西北部，玉龙乡田坝村境内，距县城140公里。大坝地理位置为：东经104°38′35″，北纬27°02′40″。属长江流域牛栏江支流。

该工程大坝为均质土坝。坝长34米，坝顶宽仅1.5米，坝高20米。迎水面为块石护坡，坝坡比为1：2，背水面为草皮护坡，坝坡比为1：2。水库总库容为2.64万立方米，集雨面积0.52平方公里。溢洪道为开敞式，进口宽4.5米，中部低，宽2米，深2.5米，长30米，开挖后未进行衬砌。放水管为D300铸铁压力管，长10米，拱涵洞长45米。

该水库大坝上半部厚度不够，但溢洪道从未溢过洪。经调洪演算，防洪标准仅达到7.7年一遇设计。如水库失事，将给下游2个村民组1500人的生命财产带来严重损害，淹没耕地930亩，其中田230亩。
