苦鱼井水库工程简介

 

苦鱼井水库位于黔西县以西18公里的锦星乡。水库集雨面积0.8平方公里，均质土坝，最大坝高18.8米，总库容12万立方米。设计灌溉面积1000亩,有效灌溉面积950亩。干渠2条长5公里。

水库校核洪水50年一遇，流量5.1立方米/秒；左岸开敞式溢洪道，坝址为土基。

1975年动工，当年建成，国家补助1.5万元。

该水库防洪未建标准，所以汛期限蓄渡汛，今后应对溢洪道改扩建。
