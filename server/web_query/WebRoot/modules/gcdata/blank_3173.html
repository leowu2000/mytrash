毛栗水库工程简介

 

毛栗水库位于大方县西面30公里的文阁乡关门山。水库集雨面积9.5公里，坝型为粘土斜墙土坝，最大坝高31.7米，另有付坝一座(土坝)，坝高23米，总库容375万立方米，干渠1条长11.4公里，支渠7条长24.4公里，设计灌溉面积10000亩，实际灌溉面积3230亩。利用库水修建的双山人畜饮水工程，解决9515人，3794头牲畜的饮水困难。渠道跌水电站一座，装机325千瓦。

水库设计洪水为50年一遇，流量62.1立方米/秒；校核洪水为300年一遇，流量74.4立方米/秒，利用右岸垭口建开敞式溢洪道泄洪。坝址为页岩，泥质灰岩，库区植被良好，库盆封闭，地下水源丰富。

工程于1957年动工，1958年春建成，坝高19.8米，蓄水84万立方米，1963年进行渠系配套，1976年大坝扩建至现有规模，1983年至1992年又对大坝进行防渗衬砌，累计工程总投资199.7万元
