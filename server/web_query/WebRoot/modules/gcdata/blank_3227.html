川洞水库工程简介

 

川洞水库位于金沙县东面35公里的安底镇化桶村。系文家桥水库右干渠尾的结瓜水库，水库集雨面积3.3平方公里，浆砌石单曲拱坝，最大坝高15.6米，总库容205万立方米，设计和有效灌溉面积均达3000亩，干渠3条长16公里，支渠7条长25公里。

水库设计洪水30年一遇，流量26.6立方米/秒；校核洪水200年一遇，流量40.2立方米/秒，坝顶溢流，溢流长度20米，坝址为寒武系白云岩。

1974年动工，1975年建成，1982年对水库进行帷幕灌浆处理和渠道防渗衬砌，累计工程总投资63.3万元。
