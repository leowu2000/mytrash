公鸡寨水库简介

   公鸡寨水库简介

         公鸡寨水库位于赫章县城西面20公里的水塘乡哑巴山，属六冲河上游。流城面积2.7平方公里。

     该水库为均质土坝，坝高27.6米，坝顶长200米，总库容96万立方米，其中防洪库容12万立方米。　
溢洪道为开敞式，其尺寸为：宽5米，过水深度0.5米，泄洪长50米。　
     放水设备为坝后闸阀控制，即放水涵管为500mm钢管，采用同型号500mm闸阀控制。
     该工程于1965年秋季动工兴建，1979年正式完工，1980年进行渠道防渗，并投入使用。
     该水库自完工运行以来，大坝处于稳定状态，溢洪道畅通，稳定，放水设备良好。由于该水库积雨面积小（2.7km2），并且整个库区森林复盖良好，故洪峰流量不大，防洪库容及校核库容均达标准，故洪水对大坝不会造成威胁，汛期较为安全的水库。该水库不存在防洪安全问题，除非遇特大暴雨，出现超标准洪水。
