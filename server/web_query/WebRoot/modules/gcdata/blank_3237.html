史家田水库位于金沙县城以东28公里的禹谟镇新民村，距金沙县城40公里，系偏岩河流域一支流。地理位置东经106°24′54″，北纬27°22′12″，有一条乡村公路通过。该水库为一天然洼地，经封堵漏水洞，开挖一条放水隧洞而成。水库集雨面积1.35km2，水库总库容93万m3，防洪库容24万m3，属小二型水库，是一座以灌溉为主的水利枢纽工程。设计灌面2500亩，旱地浇灌1500亩。

水库工程枢纽由大坝、放水系统等组成。水库总库容93万m3，属小二型水库工程，洪水标准按20年一遇设计，100年一遇校核。该水库为一天然洼地，经封堵漏水洞，开挖一条放水隧洞而成，放水设施为0.3×2m拱涵。
