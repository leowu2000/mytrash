水库简介

金沙县倒座岩水库基本情况

    金沙县为贵州省毕节地区所辖县之一，位于贵州省西北部、毕节地区东北部，地理坐标东经105°47′～106°44′，北纬27°07′～27°46′之间，境内东西长100km，南北宽50km，政区面积2528km2。县域东以偏岩河为界和遵义相望，南与息烽、修文和黔西相交，西与金沙相接，北与仁怀和四川古蔺等各县、市接壤。县城距贵阳市242km，距毕节地区155.8km，距遵义市123.3km。境内交通主要有326国道线通过，在26个乡镇中，70%以上的村通公路，里程达807.4km，在乌江度水库建成以后，为金沙县内增添了156km的航道线路，现各乡镇均有乡级公路相通，交通较方便。

全县现辖7镇19乡559个村民组共60.84万人（据2007年底统计）,其中农业人口53.01万人，居住有汉、苗、彝、布依、满、仡佬、水族、回族、侗族等14个民族。

岩孔镇位于金沙县城西北部，毕节地区东北部，周边主要有金沙县城、西洛河、平坝、桂花乡，以及遵义仁怀市的五马镇和泮水镇。岩孔镇距金沙县城21km，距毕节市区146km，全镇下辖14村共41400人，总面积125km2，耕地面积32278亩，其中农田18478亩，土地13800亩，财政收入75万元，粮食总产量1508t，农业总产值7554.81万元，农民人均年收入2361元，人均占有粮食450kg。境内交通主要有岩孔至金沙县城、岩孔至遵义仁怀市五马镇，以及岩孔至清池镇的乡级公路相通。

倒座岩水库地处毕节地区金沙县岩孔镇箐丰村境内，位于金沙县西北部，岩孔镇西部，坝址距箐丰村300m，地理坐标东经106°06′25″，北纬27°33′38″。水库所处河流为长江流域乌江水系偏岩河干流的一级支流箐河河流。流域内多年平均气温15℃，日照时数1095h，多年平均径流量360.5万m3，平均降雨量为1034.2mm。坝址处集雨面积27.6km2（其中上游箐河水库区间8.9km2）,主河道长7.6km（其中上游箐河水库区间3.8km），平均比降45.1‰。水库设计洪水标准为30（p=3.33%）年一遇，相应洪峰流量为168m3/s，下泄流量166 m3/s，库容为104万m3，校核洪水标准为200（p=0.5%）年一遇，相应洪峰流量为263 m3/s，最大下泄流量249 m3/s，库容为109万m3。

倒座岩水库距上游箐河水库5.5km，距下游西堰坎水库20km；距岩孔镇18km，距金沙县城39km，现有岩孔至清池镇的乡级公路由坝下300m处箐丰村通过，工程区交通较方便。水库现坝顶高程1082.52m，坝底见基面高程1056.10m，最大坝高26.42m，大坝为浆砌石单曲拱坝，坝顶宽2.65m，坝顶长82m，其中未拆除坝段长58m，临时度汛缺口长24m，坝底最大宽度6.0m。临时度汛缺口位于大坝左端，为倒梯形结构，上口宽24m，下口宽9m，底部高程1073.57m。坝体中段底部设置φ500冲沙管，进口中心高程1060.85m，末端设φ500闸阀1个；左段坝体底部设有φ600放水管，进口中心高程1068.30m，末端设有1个φ600工作闸阀和1个φ600事故闸阀。

水库坝址出露岩石主要为寒武系下统金顶山组岩层，岩石较风化，两坝肩第四覆盖层较厚，且左坝肩1081.20m高程灰岩地层内发育有一条宽度为1.4m的溶蚀裂隙，长60.17m，深16.07m，坝肩岩体处于临空不稳定状态，右坝肩接近坝顶高程处坝基为砂页岩强风化层和含碎石粘土层上，两坝肩地质条件较差。

倒座岩水库是以灌溉、防洪为主，并兼发电的小(1)水库，水库原设计灌溉面积6000亩，防洪保护下游箐口乡、岩孔镇12000人和11000亩耕地，原设计发电装机容量2×75kw。该水库为一未完建工程，水库现为一病险水库，且现有规模不满足设计灌溉及防洪要求，为达到工程效益，倒座岩水库在除险加固时，需恢复到原设计状况并适当加高坝体。根据金沙县人民政府文件(金府涵［2007］39号)精神，金沙县政府向省水利厅承诺，除险加固时恢复到原设计坝顶高程所需工程投资由金沙县人民政府出资，该部分投资不计入倒座岩水库除险加固工程总投资中，承诺函见附件：金沙县人民政府文件。

倒座岩水库工程自停建以来，水库一直处于空库运行，未发挥工程效益。现大坝应力及拱座稳定不满足规范要求，本次除险加固对坝体上下游面进行加高加厚处理，并在左右坝肩设置1、2＃重力墩，同时对坝体采取防渗补强帷幕灌浆，以及对其岩石中风化裂隙、节理裂隙及灰岩中溶蚀裂隙进行固结灌浆等措施。

工程总投资618.69万元，施工总工期为9个月。
