申家沟水库工程简介

 

申家沟水库位于赫章县西北25公里的六曲河镇和平村。水库集雨面积3.6平方公里，均质土坝，最大坝高27.6米，总库容58万立方米，设计灌溉面积1200亩，干渠1条长11公里。

水库设计洪水20年一遇，流量11立方米/秒；校核洪水200年一遇，流量15.8立方米/秒，左岸开敞式溢洪道泄洪。

1964年动工，1966年建成，总投资16.3万元。因渠道年久失修，垮塌、渗漏严重，灌溉面积逐年下降，至91年仅灌田40亩。
