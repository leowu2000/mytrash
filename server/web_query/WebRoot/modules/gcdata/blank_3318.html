石门坎水库工程简介

 

石门坎水库位于赫章县以东35公里的野马川镇石垭村。水库集雨面积0.9平方公里，均质土坝，最大坝高18米，总库容33万立方米，设计灌溉面积500亩,干渠1条长4.6公里。

水库设计洪水200年一遇，流量5.7立方米/秒；右岸开敞式溢洪道。坝址为土基。

1976年动工，1981年建成，总投资5.7万元。该水库因坝身及坝肩漏水，溢洪道开挖断面不够，无衬砌，放水设备漏水等原因而不能蓄水，故无灌溉效益。
