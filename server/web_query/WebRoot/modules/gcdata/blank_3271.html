工程简介

织金县《险峰水库防御洪水（应急）预案》是依据《中华人民共和国防洪法》、《中华人民共和国水法》、《水库大坝安全管理条例》等国家有关法律法规，同时综合考虑《综合利用水库调度通则》、《水库管理通则》、《蓄滞洪区安全建设指导纲要》等规则，结合险峰水库下游现状和水库实际情况进行编制。本预案是为进一步提高险峰水库应对突发事件的能力，做到有组织、有计划地处置突发事件，最大限度地减少伤亡和财产损失，维护社会稳定而编制的。在完成应急预案编制的同时，还应积极做好宣传，并适时组织演练，做到未雨绸缪，有备无患。

根据该水库工程设施现状、库区地质情况及下游防洪设施情况综合分析，在水库大坝出现漫溢、散浸和管涌、大坝裂缝和渗水险象、大坝滑动和倾覆险象、放水闸门故障及下游危急险象、人为破坏等危及大坝安全的恐怖事件、发生超过水库校核标准的洪水等任意一种情况时，由水库防洪应急指挥部根据发生的危急险情认真分析后向指挥长提出申请启动预案，由指挥长下令启动本防洪应急预案。
